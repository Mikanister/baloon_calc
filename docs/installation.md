# Встановлення

## Вимоги

- Python 3.8 або вище
- Windows, Linux або macOS

## Встановлення з pip

```bash
pip install -r requirements.txt
```

## Встановлення як пакет

```bash
pip install -e .
```

## Залежності

Основні залежності:
- `matplotlib` - для графіків
- `numpy` - для математичних обчислень
- `pydantic` - для валідації
- `ttkbootstrap` - для UI
- `pandas` - для експорту даних
- `plotly` - для 3D візуалізації
- `scipy` - для оптимізації
- `rich` - для консольного виводу

## Запуск exe файлу

Для Windows доступний готовий exe файл:
- `dist_exe/v2.0/baloon_calculator.exe`

Просто запустіть файл подвійним кліком.
