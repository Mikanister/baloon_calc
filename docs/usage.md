# Використання

## Основні функції

### Розрахунок параметрів

1. Виберіть режим розрахунку:
   - **Об'єм ➜ навантаження** - розраховує навантаження для заданого об'єму
   - **Навантаження ➜ об'єм** - розраховує необхідний об'єм для заданого навантаження

2. Введіть параметри:
   - Тип газу (Гелій, Водень, Гаряче повітря)
   - Матеріал оболонки
   - Товщина оболонки
   - Висоти пуску та польоту
   - Форма кулі та її параметри

3. Натисніть "Розрахувати"

### Експорт результатів

- **Експорт результатів** - зберігає результати в Excel файл
- **Експорт викрійки** - зберігає викрійку в PNG або Excel формат
- **3D модель** - показує інтерактивну 3D модель кулі

### Збереження налаштувань

Натисніть "Зберегти налаштування" для збереження поточних параметрів.

Налаштування автоматично завантажуються при наступному запуску.

## Підтримувані форми

- **Сфера** - класична форма
- **Подушка/мішок** - прямокутна форма
- **Груша** - конічна форма з округленими краями
- **Сигара** - циліндр з округленими кінцями
