# API: Розрахунки

## `calculate_balloon_parameters`

Основний функціонал для розрахунку параметрів аеростата.

### Параметри

- `gas_type`: Тип газу ("Гелій", "Водень", "Гаряче повітря")
- `gas_volume`: Об'єм газу (м³) або бажане навантаження (кг)
- `material`: Матеріал оболонки
- `thickness_mm`: Товщина оболонки (мкм)
- `start_height`: Висота пуску (м)
- `work_height`: Висота польоту (м)
- `ground_temp`: Температура на землі (°C)
- `inside_temp`: Температура всередині (°C)
- `mode`: Режим розрахунку ("payload" або "volume")
- `duration`: Тривалість польоту (год)
- `shape_type`: Форма кулі ("sphere", "pillow", "pear", "cigar")
- `shape_params`: Параметри форми (словник)

### Повертає

Словник з результатами розрахунку:
- `required_volume`: Необхідний об'єм (м³)
- `surface_area`: Площа поверхні (м²)
- `envelope_mass`: Маса оболонки (кг)
- `lift`: Підйомна сила (Н)
- `payload`: Корисне навантаження (кг)
- та інші параметри
