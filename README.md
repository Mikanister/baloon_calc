# Калькулятор аеростатів

Сучасний калькулятор для розрахунку параметрів аеростатів (повітряних куль) з графічним інтерфейсом на Tkinter.

## Можливості
- Розрахунок підйомної сили, маси оболонки, втрат газу, напруги, об'єму тощо
- Фізично обґрунтована модель втрат газу через оболонку
- Інтерактивна валідація та підказки до всіх полів
- Темна тема, сучасний UI: розділення вікна на форму (зліва) та результати зі скролом (справа), статус-бейдж (ок/попередження/критично)
- Побудова графіків залежностей (matplotlib) з маркерами ключових точок і орієнтирною лінією підйомної сили на м³
- Підтримка різних матеріалів та газів, різних форм оболонки (сфера/циліндр/тор/подушка), збереження налаштувань

## Готові релізи (exe)

Усі стабільні версії програми для Windows зберігаються у папці `dist_exe/`:
- `dist_exe/v2.0/baloon_calculator.exe` — актуальний UI з розділенням екрану і покращеними графіками
- `dist_exe/v1.0/baloon_calculator.exe` — перший стабільний реліз (без нових UI-покращень)

### Запуск exe-файлу
1. Перейдіть у папку відповідної версії, наприклад, `dist_exe/v2.0/`
2. Запустіть файл `baloon_calculator.exe` подвійним кліком
3. Python не потрібен

## Запуск з Python (альтернатива)
1. Встановіть Python 3.8+
2. Встановіть залежності:
   ```
   pip install -r requirements.txt
   ```
3. Запустіть:
   ```
   python -m baloon.main_improved
   ```
   або напряму:
   ```
   python baloon/gui.py
   ```

## Запуск тестів
1. Встановіть залежності (включає pytest):
   ```
   pip install -r requirements.txt
   ```
2. Запустіть всі тести:
   ```
   pytest
   ```
   або з покриттям коду:
   ```
   pytest --cov=baloon --cov-report=html
   ```
3. Є базові юніт-тести (зокрема `_compute_lift_state`); детальніше про тести див. `tests/README.md`
